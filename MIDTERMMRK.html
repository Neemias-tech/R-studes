Mid Term - exploratory analysis of GPA

This is an exploratory analysis of a data set GPA, from a survey from Duke University where 55 students asked about their GPA, number of hours they study at night, number of nights they go out, and their gender.

The final goal of this project its to create a model to predict the GPA based in one or more variables.

The first observation before we start the analysis is this data sample is small for the population being observed. So, my first recommendation is about increasing the number of students being asked to at least 100, to have a better analysis and with a higher confidence level.
